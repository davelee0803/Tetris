# Tetris - 俄羅斯方塊網頁版

一個使用 HTML + CSS + JavaScript 實作的經典俄羅斯方塊遊戲，可直接在瀏覽器中遊玩。

## 功能

- 7 種經典方塊（I, O, T, S, Z, J, L）
- 鍵盤控制（方向鍵 / WASD）
- Ghost piece 預覽落下位置
- 下一個方塊預覽
- 分數、等級、消除行數顯示
- 等級越高，掉落速度越快
- Game Over 偵測與重新開始

## 控制方式

| 按鍵 | 動作 |
|------|------|
| ← / A | 向左移動 |
| → / D | 向右移動 |
| ↓ / S | 加速下落 |
| ↑ / W | 旋轉方塊 |
| Space | 直接落下 |

## 遊玩方式

直接在瀏覽器打開 `tetris.html` 即可開始遊戲。
